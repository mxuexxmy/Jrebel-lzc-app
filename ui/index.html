离线激活指南

1
生成专属 GUID
使用 GUID 生成器 或任意 UUID 工具生成一串随机字符串。
2
配置激活地址
在 IDEA 的 JRebel 设置中，找到 License Server URL 并填入以下地址（请确保激活服务已启动）：
http://127.0.0.1:12345/{你的GUID}
提示：邮箱栏填写任意有效格式的邮箱即可。
3
在线激活
选择在线模式 (Activate online) 并点击 Activate。激活完成前，请保持激活服务运行且不要关闭命令行窗口。
4
切换离线模式
激活成功后，在 JRebel 设置中点击 Work Offline 切换为离线模式。之后即可关闭激活服务。
